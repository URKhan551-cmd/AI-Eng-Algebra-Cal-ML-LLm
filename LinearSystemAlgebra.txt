// Variable: Unknown value (x, y, z)
// Linear equation: Variables have power 1 only.
// Linear system: Two or more linear equations solved together.
// Solution: Values that satisfy all equations simultaneously.
// Consistent system: Has at least one solution.
// Inconsistent system: Has no solution.
// Unique solution: Exactly one answer.
// Infinite solutions: Infinitely many answers.

Phase 4 — Linear Systems ⭐⭐⭐⭐⭐

A linear system is simply multiple linear equations solved together.
Example
x + y = 7
2x + y = 11
Graphically, each equation is a line. The solution is the point where the lines intersect.
